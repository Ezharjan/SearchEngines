 # Polymerized Search  多元聚合搜索引擎

​	





## Customized: 自定义

```bash
cd src/
vi config.js
...(customized as you wish)
```

## Development: 本地使用
```bash
npm install
npm run start
```

## Production : 生产环境使用

```bash
npm run build
```
